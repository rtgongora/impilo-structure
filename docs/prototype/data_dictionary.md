# Impilo vNext — Data Dictionary

> Schema-accurate reference for all tables surfaced in the prototype UI.
> Derived from `src/integrations/supabase/types.ts` (auto-generated from live schema).

---

## Conventions

- **PK**: Primary key (uuid `id` unless noted)
- **Req**: Required on insert (no default)
- **Opt**: Has default or nullable
- **FK**: Foreign key reference
- **RLS**: Row-Level Security notes (high-level)
